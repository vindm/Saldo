# Group Profile — Direct

These are our own clients; we own the relationship end to end. A client's own `profile.md` overrides any line here.

## Communication
- We talk to the client directly, usually on Telegram.
- No intermediary; we set expectations and deadlines ourselves.

## Data & connectors
- We pull bank statements ourselves (T-Bank / Alfa).
- Fiscal/cash data via OFD where the client runs a cash register.

## Tone
- Friendly and plain-spoken; always tell the client exactly what to do next.
