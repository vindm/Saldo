# Indonesia (id) — operator glossary

Source of glosses for **operator-facing** text per `policies/INSTRUCTIONS.md §0.1`. When you
write a task/calendar/message for an Indonesian client served by a non-Indonesian operator,
keep the real term but attach the plain explanation + the operator-system analogy. Analogies
below are written for a **Russian-jurisdiction** operator (the practice's locale).

| Term | Plain meaning | RU operator analogy |
|---|---|---|
| **PPh** | Pajak Penghasilan — income tax (family of taxes) | налог на доход |
| **PPh 21** | income tax withheld from employee salaries by the employer | подоходный с зарплат, как НДФЛ |
| **PPh 23** | withholding on payments for services to contractors | удержание налога с оплаты услуг |
| **PPh 4(2) / final (sewa)** | final tax on rent, 10%, withheld by the tenant | налог с аренды (удерживает арендатор) |
| **PP55 / UMKM final / "0.5%"** | small-business final tax: 0.5% of gross turnover | льготные 0,5% с оборота, как УСН «доходы» |
| **PPh Badan** | corporate income tax, 22% (after the 0.5% facility lapses) | налог на прибыль компании |
| **PPN** | Pajak Pertambahan Nilai — VAT, 11% | НДС |
| **PKP** | VAT-registered taxpayer status (threshold 4.8B turnover) | статус плательщика НДС |
| **BPJS** | the two employer social-insurance contributions (health + employment) | взносы за сотрудников (медстрах + соцстрах) |
| **BPJS Kesehatan** | employee health-insurance contributions, 5% of wage | взносы на медстраховку, как ОМС |
| **BPJS Ketenagakerjaan** | employment social-security contributions | взносы на соцстрах, как СФР |
| — **JHT / JKK / JKM / JP** | old-age savings / work-accident / death / pension | накопительная / от травм / на случай смерти / пенсия |
| **TER** | the ready monthly effective rate for computing PPh 21 | готовая месячная ставка подоходного |
| **kode billing** | a payment code/slip to pay a specific tax | платёжка-квитанция на конкретный налог |
| **NTPN** | the receipt number proving a tax was paid | квитанция об уплате налога |
| **Coretax** | the tax authority's online portal (login/file/pay) | личный кабинет налоговой, как ЛК ФНС |
| **DJP** | Direktorat Jenderal Pajak — the tax authority | налоговая служба, как ФНС |
| **SPT Masa** | the monthly tax return | ежемесячная декларация |
| **SPT Tahunan** | the annual tax return (Badan = company) | годовая декларация |
| **unifikasi** | one combined billing for withholdings (PPh 4(2)+23) | единая платёжка по удержаниям |
| **bukti potong** | the withholding certificate issued to a counterparty | справка об удержанном налоге |
| **LKPM** | quarterly investment-activity report (OSS) | квартальный инвест-отчёт |
| **NPWP** | the company's tax number (16 digits) | ИНН компании |
| **NIB / OSS** | business identity number / business-licensing system | как ОГРН / регистрация бизнеса |
| **Moka** | the salon's POS/cash app (source of sales) | касса салона (учёт продаж) |
| **peredaran bruto** | gross turnover (the 0.5% tax base) | валовая выручка (база налога) |
| **KAP-KJS** | the tax/payment-type code inside a kode billing | как КБК |
| **PTKP** | annual non-taxable income threshold for PPh 21 | необлагаемый минимум по зарплате |
| **NIK** | the employee's national ID number (links to NPWP) | как СНИЛС/паспортные данные сотрудника |
| **EDABU** | the BPJS Kesehatan employer portal (member roster) | портал медстраховки (реестр сотрудников) |
| **EPS / SIPP** | the BPJS Ketenagakerjaan employer portal/payment | портал соцстраха (реестр и оплата) |
